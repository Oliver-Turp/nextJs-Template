This is my Next.js template project.

## Getting Started

1. Create your app
```bash
npx create-next-app your-app-name --example https://github.com/Oliver-Turp/nextJs-Template
```

2. Install the dependencies
```bash
npm run install
```

3. Check for updates
```bash
ncu
```
```bash
ncu -g
```

4. run the development server:
```bash
npm run dev
```
